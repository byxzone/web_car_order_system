### 驾校智能化管理系统

#### 1.介绍

本系统使用PHP作为后端、MUI作为前端框架开发。实现了驾校学员报名学车、约车训练、更改预约、评价教练，驾校教练发布约车计划、查看某个计划里约车的学员、学员管理等功能。

本系统是我编写的第一个功能较为完整的Web系统，代码比较稚嫩，望大家包容。

#### 2.文件结构说明

../api/   ---- 后端程序，主要为操作数据库与实现核心功能的接口函数

../css/  ---- 前端框架的CSS样式文件

../fonts/ ---- 前端框架的支持字库

../img/ ---- UI界面中的图片

../js/ ---- 实现前端功能的JS文件

../subpage/ ----各个子页面

#### 3.数据库字段说明

user 用户表

 id 用户id

 username 登录名

 name 用户姓名

 password 密码(MD5)

 phone 电话号码

 type 类型(0为学员/1为教练)

 coach 归属教练(若本身用户类型为教练此项为空)

class 预约模板表

 id 模板id

 coach 教练id

 time_begin 开始时间(文本格式，如09:00)

 time_end 结束时间(文本格式，如11:00)

 number 可预约学员总数

reserve 预约订单表

 id 预约订单id

 template int型，对应class表中模板id

 date 日期，文本型，精确到日，03-18

 coach 教练id

 user 学员id

 stat 订单当前状态：0已预约未进行/1已预约已完成/2已预约已取消
